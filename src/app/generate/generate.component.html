Was macht ein Sieve Skript?

Ein Sieve Skript sortiert Emails, auf diese Weise können Emails sortiert, markiert oder gelöscht werden.

Dieser Sieve Skript: Markiert Emails wichtiger Kontakte Blockt Emails größer als {{ value }}MB Löscht Emails welche bestimmte Worte enthalten
Wie generiere ich einen Sieve Skript?

Unser Tool kann Dir helfen, ganz leicht einen Sieve Skript zu erstellen.
Gib einfach die Email Adressen Deiner Klassenleitung ein und der Sieve Skript wird runtergeladen.
